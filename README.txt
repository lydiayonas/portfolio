Lydia Yonas Portfolio - File Structure
=======================================

index.html          Main HTML file
css/
  base.css          CSS variables, reset, typography, layout
  background.css    Animated canvas background + cursor glow styles
  nav.css           Navigation bar + buttons
  hero.css          Hero section, photo card, reveal animation, marquee strip
  sections.css      About, Skills, Projects, Certs, Contact, CTA, Footer
  responsive.css    All media queries + hamburger menu
js/
  background.js     Animated particle canvas background
  cursor.js         Cursor glow follow effect
  reveal.js         Scroll reveal (IntersectionObserver)
  nav.js            Nav scroll shadow + hamburger toggle
